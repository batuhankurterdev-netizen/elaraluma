# Karakter Görselleri

Bu klasöre aşağıdaki görselleri ekleyin:

1. **elara.png** - Elara karakterinin görseli
   - Turuncu, kıvırcık saçlı küçük kız
   - PNG formatında
   - Önerilen boyut: 512x512 veya daha büyük

2. **luma.png** - Luma karakterinin görseli  
   - Siyah, büyülü kedi
   - Yıldız tozları efekti
   - PNG formatında
   - Önerilen boyut: 512x512 veya daha büyük

## Görselleri Nasıl Eklerim?

1. Görselleri bu klasöre (`assets/images/`) kopyalayın
2. Dosya isimlerinin tam olarak `elara.png` ve `luma.png` olduğundan emin olun
3. Uygulamayı yeniden çalıştırın: `flutter run`

Görseller yoksa otomatik olarak emoji gösterilecektir.
